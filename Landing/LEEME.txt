Pasos para crear los servicios


1. se copia primero el composer.json y por consola se genera el codigo:
composer update

ejemplo: 
~/workspace/General/loginUsuario $ composer update

2. el automaticamente crea la carpeta vendor, pero se debe copiar el index.php, la carpeta template.

3. para correr la prueba se utiliza el siguiente comando:
vendor/bin/phpunit --bootstrap vendor/autoload.php Test.php
